# Sales Data Analysis Using Excel

## Project Overview
This project focuses on analyzing retail sales data using Microsoft Excel. The goal was to explore transactional data, identify trends, and generate actionable insights related to revenue, customer behavior, and product performance.

---

##  Objectives
- Clean and prepare raw sales data
- Analyze monthly revenue trends
- Evaluate product category performance
- Understand customer segmentation
- Generate key business insights

---

##  Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Excel Functions (SUM, COUNTA, XLOOKUP)
- Data Cleaning & Formatting

---

##  Dataset Description
The dataset includes the following fields:
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount (Revenue)

---

##  Analysis Performed

### 1. Monthly Revenue Trend
- Aggregated revenue by month using Pivot Tables
- Identified variations in sales performance across months

### 2. Product Category Analysis
- Analyzed revenue contribution by each category
- Identified top-performing categories

### 3. Customer Segmentation
- Compared revenue distribution by gender
- Analyzed customer behavior patterns

### 4. Key Metrics (KPIs)
- Total Revenue
- Total Quantity Sold
- Total Transactions
- Average Transaction Value
- Top Category

---

## Key Insights
- Revenue fluctuates across months, indicating demand variability
- Certain product categories dominate total sales
- Customer purchasing behavior differs across segments
- Seasonal patterns may influence sales performance

---

##  Conclusion
This project demonstrates how Excel can be effectively used for data analysis and reporting. By leveraging Pivot Tables and formulas, raw data was transformed into meaningful business insights.
